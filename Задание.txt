Тестовое задание:

Описание задачи:
Необходимо разработать систему для управления реферальной программой школы.

Требования:

1. Генерация ссылки для приглашения учеников в школу:
• Создать endpoint для генерации уникальной ссылки приглашения.
• Ссылка должна содержать уникальный идентификатор реферера.
2. Регистрация ученика по приглашению:
• Создать форму для регистрации ученика с полями: ФИО, телефон, email.
• Сохранить информацию о реферере после заполнения формы.
3. Обработка оплаты:
• Создать endpoint для добавления информации об оплате.
• Добавить по 4 урока для обоих студентов (реферера и приглашенного).
4. Статистика приглашенных учеников:
• Вести учет приглашенных учеников.
• Создать endpoint для получения статистики по приглашенным ученикам.

Дополнительные требования (опциональные):

1. Валидация данных:
• Реализовать валидацию данных на всех этапах (регистрация, оплата и т.д.).
2. Аутентификация и авторизация:
• Добавить JWT аутентификацию для всех защищенных endpoint'ов.
3. Логирование и мониторинг:
• Реализовать логирование действий пользователей и ошибок.
• Настроить мониторинг состояния системы.
4. Кэширование:
• Использовать Redis для кэширования часто запрашиваемых данных.
5. Тестирование:
• Написать unit и integration тесты для основных функциональностей.

Ожидаемые результаты:
• Исходный код проекта на GitHub или другом репозитории.
• Инструкции по запуску и тестированию проекта.
• Документация по API.
